Initialize a new Expo React Native project. We are building a hypercasual, high-performance brain-teaser game called "Signal vs Noise" (Sinyal Gürültü). Use a clean, modular S-Class architecture. 

Core Concept: Words continuously fall from the top of the screen to the bottom. The player must tap ONLY the "Signal" (correct words) and ignore the "Noise" (leetspeak/fake words).

Game Mechanics & Rules:
1. Word Pools: 
   - Signals (Valid): 'KEDİ', 'KÖPEK', 'TAVŞAN', 'ASLAN', 'KAPLAN', 'KARTAL'
   - Noises (Traps): 'K3Dİ', 'D0G', 'KÖP3K', 'T4VŞAN', 'A5LAN', 'MASA', 'ELMA'
2. Spawning: Words spawn at the top at random X-coordinates and fall straight down. As the score increases, spawn frequency increases, falling speed increases, and the ratio of "Noise" words gets higher.
3. Interactions (CRITICAL):
   - Tap a Signal: Score +1, word disappears, trigger light haptic feedback.
   - Tap a Noise: Instant GAME OVER, trigger heavy error haptic feedback.
   - Let a Signal fall off the screen: Instant GAME OVER.
   - Let a Noise fall off the screen: Safe (nothing happens, it just disappears).

Performance & Architecture (Crucial for 60fps):
- Do NOT use React state for frame-by-frame falling. Use `react-native-reanimated` for the falling animations (e.g., animating the Y translation from top to bottom). Each falling word should ideally be its own memoized component managing its own animation.
- Implement `expo-haptics` for tactile feedback.

UI & Aesthetic (Sleek, Dark Mode):
- Background: Pure black (`#000000`).
- Text: Sleek, monospace or modern font. All falling words should look visually identical in style (e.g., pure white or neon green) so the player must read them, not rely on colors.
- Top Header: Display the current Score elegantly. 
- Game Over Screen: A striking, semi-transparent dark overlay with the final score, a "Best Score" tracker (use AsyncStorage), and a spring-animated "Play Again" button.

Please write the complete logic, hooks (e.g., `useSignalGame`), UI components, and wire everything up into a production-ready prototype in one go.
